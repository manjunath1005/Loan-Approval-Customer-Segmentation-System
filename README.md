# 🏦 Loan Approval & Customer Segmentation System

An end-to-end Machine Learning application that helps financial institutions evaluate loan applicants, identify customer segments, and visualize customer behavior using Logistic Regression, K-Means Clustering, and PCA.

## Features

- 💰 Loan Approval Prediction
- 📊 Default Risk Assessment
- 👥 Customer Segmentation
- 📉 PCA-Based Visualization
- 🌐 Interactive Streamlit Dashboard

## Technologies Used

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy
- Joblib

## Machine Learning Models

- Logistic Regression
- K-Means Clustering
- Principal Component Analysis (PCA)

## Dataset

- 255,347 Loan Records
- 16 Features
- Binary Loan Default Classification

## Results

- Logistic Regression Accuracy: 67.4%
- Default Recall: 70%
- 3 Customer Segments Generated
- PCA Variance Retained: 40.16%
